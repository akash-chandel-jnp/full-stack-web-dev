Auto Grid Sizing

Auto grid sizing for columen here means first and second column have Fixed sizing of 300px and 200 px , while the third row defined using auto , will take up the remaining space.

But if row is defined using auto then it does not take up the whole remaining space below it, rather it takes only the space required for its content height
